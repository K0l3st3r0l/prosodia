# Marco Pedagógico — ProsodIA

## Fórmulas de cálculo

- **% errores** = (errores totales / palabras del texto) × 100
- **% precisión** = 100 − % errores
- **Velocidad cruda** = palabras leídas / minutos
- **Velocidad correcta (PCPM)** = palabras correctas / minutos
- **Índice compuesto** = 40% precisión + 30% velocidad + 30% prosodia

## Niveles de logro — Velocidad

| Nivel | Descripción |
|-------|-------------|
| Muy Lenta | Muy por debajo del estándar |
| Lenta | Bajo el estándar |
| Medio Baja | Ligeramente bajo |
| Medio Alta | Dentro del estándar |
| Rápida | Sobre el estándar |
| Muy Rápida | Muy sobre el estándar |

## Niveles de logro — General

- Muy Bajo lo Esperado
- Bajo lo Esperado
- Lo Esperado

## Calidad de lectura

- `silábica` — lee sílaba a sílaba
- `palabra_a_palabra` — lee palabra por palabra con pausas
- `unidades_cortas` — lee en grupos pequeños
- `fluida` — lectura natural y continua

## Prosodia

Evaluada según criterios Mineduc / Agencia de Calidad de la Educación Chile:
- Entonación adecuada al tipo de texto
- Pausas respetando puntuación
- Ritmo natural del habla
- Expresividad acorde al contenido
